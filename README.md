## Portafolio – Ingeniero de Sistemas

Portafolio para demostrar habilidades como **Ingeniero de Sistemas** con foco en:

- **Backend**: Node.js
- **Frontend**: Angular y React

La aplicación actual es un **portafolio en React** que presenta tus skills y proyectos, con secciones dedicadas a Node.js, Angular y React.

### Scripts

- **Instalar dependencias**:

```bash
npm install
```

- **Entorno de desarrollo**:

```bash
npm run dev
```

Luego abre `http://localhost:5173` en tu navegador.

- **Build producción**:

```bash
npm run build
```

### Estructura principal

- `src/App.tsx`: estructura principal de la página (hero, proyectos, skills, sobre mí, contacto).
- `src/styles.css`: estilos con un diseño moderno, oscuro y responsivo.

### Cómo personalizarlo

- **Datos personales**: cambia el texto de la sección `Sobre mí` y el nombre del footer en `App.tsx`.
- **Proyectos**: edita el arreglo `PROJECTS` en `App.tsx` con tus propios proyectos de Node.js, Angular y React, añadiendo enlaces reales a GitHub o demos.
- **Contacto**: sustituye el formulario maquetado por tu backend en Node.js, un servicio de email o un simple enlace a tu correo/LinkedIn.

Con estos cambios podrás mostrar claramente tu experiencia en **Node.js**, **Angular** y **React** en un solo portafolio.
